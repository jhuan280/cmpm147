Experiment 01 - Getting Started

Description

This project was inspired off the meme trend that was popular back in the late 2010's where various news articles would display headlines featuring a template along the lines of, "Florida Man (inserts crazy headline)".

Example of a Florida Man article: https://www.cbsnews.com/miami/news/drunk-florida-man-crashes-lawn-mower-police-car/

Technical

The way this program works is that it takes a random key in an array and displays it accordingly to where the array is called in the code. (recognized through the $ sign)

This process is done through the roll button that randomly chooses a word in a set of arrays and displays it in different parts of the template where the words are called.
